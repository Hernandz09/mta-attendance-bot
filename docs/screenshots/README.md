# Capturas para el README

Coloca aquí las imágenes del producto (PNG o JPG) con estos nombres exactos para que el README las muestre:

| Archivo | Qué capturar | Tips |
|---------|----------------|------|
| `01-panel-botones.png` | Mensaje fijado del bot con los botones **Marcar Entrada** / **Marcar Salida** | Canal `#asistencias`, sin datos personales sensibles |
| `02-confirmacion-entrada.png` | Respuesta efímera de entrada exitosa (embed) | Incluye fecha, hora y estado (Puntual/Tardanza) |
| `03-confirmacion-salida.png` | Respuesta efímera de salida exitosa | |
| `04-comando-estado.png` | Resultado de `/asistencia estado` | Puedes hacerlo en un canal donde sí haya permission de texto |
| `05-dashboard.png` | Vista del dashboard web con el registro reflejado | Recorta nombres reales si hace falta (CV público) |
| `06-google-sheets.png` | Fila de ejemplo en Google Sheets (opcional) | Oculta `discord_id` o úsalo de prueba |

## Cómo tomarlas rápido en Discord (Windows)

1. Abre Discord en el servidor de demo.
2. `Win + Shift + S` → recorta la zona → guarda en esta carpeta.
3. O captura completa y recorta con Paint / PowerToys.

Cuando las tengas, el README en la raíz ya referencia estas rutas. Haz commit de las imágenes y listo.
